## 1. 📖 项目简介

<div align="center">

[![Made with Love](https://img.shields.io/badge/Made%20with-❤️-red.svg)](https://github.com/henrry179/NoProblem)
[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg)](https://github.com/henrry179/NoProblem/pulls)

</div>

NoProblemBot 是一个专门生成积极鼓励短句的工具，旨在帮助人们应对各种生活和工作中的压力与挑战。通过中英文对照的鼓励短句，为用户提供积极的心理支持。

### 🌟 为什么选择 NoProblemBot？

- 🎯 **简单易用**：清晰的格式规范，易于理解和应用
- 🌈 **丰富多样**：涵盖多种情绪和场景的鼓励短句
- 💪 **积极向上**：每个短句都充满正能量和建设性建议
- 🌍 **双语支持**：中英文对照，满足不同语言需求
- 🎨 **视觉友好**：搭配表情符号，增强表达效果

## 3. 🚀 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/henrry179/NoProblem.git

# 进入项目目录
cd NoProblem
```

### 使用示例

```markdown
# 情绪类短句示例
No problem, although you're feeling overwhelmed with joy, but remember to stay grounded and share your happiness with others 🌟✨
没关系，虽然你被喜悦冲昏了头脑，但是要记得保持冷静，把快乐分享给他人，很厉害 🌟✨
```

## 4. 📝 短句格式规范

<div align="center">

| 序号 | 规范 | 示例 |
|:---:|:---:|:---:|
| 1 | 英文句以 "No problem," 开头 | No problem, although... |
| 2 | 中文句以 "没关系，" 开头 | 没关系，虽然... |
| 3 | 使用 "虽然...但是..." 结构 | although...but... |
| 4 | 英文句末不包含 "— 很厉害" | - |
| 5 | 中文句末保留 "很厉害" | 很厉害 |
| 6 | 配有2-3个表情符号 | 🌟✨ |

</div>
